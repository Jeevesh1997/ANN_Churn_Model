# ANN_Churn_Model
Developed a model to predict if a person is likely to stay or leave a bank given the following attributes: 
- Geography
- CCredit Score
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Does this customer have a credit card?
- Is this customer an Active Member
- Estimated Salary

The model should predict: 
- Excited?: True/False
